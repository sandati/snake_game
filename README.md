### Start game

1. Install Processing
    - [How to download processing?](https://processing.org/download)

2. Launch project
    - Clone repository: ```git@github.com:sandati/snake_game.git```

    - Open processing

    - Open project: ```file -> open...``` or ```Ctrl + O```

    - Run
